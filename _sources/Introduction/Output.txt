Output
======

``pyfem2`` writes output to a `ExodusII
<http://prod.sandia.gov/techlib/access-control.cgi/1992/922137.pdf>`__
database. ExodusII is a file format developed at `Sandia National Laboratories
<http://www.sandia.gov>`__, for storing finite results. The ExodusII output
database is a data file containing initialization, model, and results data
generated by the analysis of a finite element model, as shown

.. figure:: ExoModel.png
   :align: center

   The ExodusII data model

The results contained in the ExodusII output database can visualized and postprocessed with the open-source visualization toolkit `ParaView <http://www.paraview.org>`__.
