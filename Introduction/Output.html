pyfem2 writes output to a ExodusII database. ExodusII is a file format developed at Sandia National Laboratories, for storing finite results. The ExodusII output database is a data file containing initialization, model, and results data generated by the analysis of a finite element model, as shown

The ExodusII data model

The results contained in the ExodusII output database can visualized and postprocessed with the open-source visualization toolkit ParaView.
